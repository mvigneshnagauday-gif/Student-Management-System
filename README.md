# 🎓 Student Management System

## 📌 Project Overview

The Student Management System is a full-stack web application developed using Spring Boot, MySQL, HTML, CSS, and JavaScript. It helps manage students, departments, and user authentication.

---

## 🚀 Features

- User Registration
- User Login
- Dashboard
- Add Student
- Update Student
- Delete Student
- View Students
- Add Department
- View Departments
- MySQL Database Integration
- REST APIs
- Exception Handling
- DTO Pattern
- Layered Architecture

---

## 🛠 Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Maven

### Database
- MySQL

## ▶️ How to Run

### Backend

1. Open Backend in STS.
2. Configure MySQL.
3. Run the Spring Boot application.

### Frontend

Open `index.html` in your browser.
